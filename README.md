## 📌 Project Overview

Sip Coffee is a front-end project that simulates a real-world coffee shop website.  
It focuses on UI/UX design, responsiveness, and user interaction.

The website includes:

- Product showcase
- Shopping cart system
- Login & signup pages
- Payment UI
- Responsive layout for all devices

---

## ✨ Features

- 🏠 Modern home page UI
- 🛒 Fully functional cart system (frontend)
- 🔐 Login & Signup interface
- 💳 Payment page design
- 🎨 Clean and modern UI design
- 📱 Fully responsive (mobile / tablet / desktop)
- 🎥 Supports images & media content
