## 📌 Project Overview
Sip Coffee is a front-end project designed to simulate a real coffee shop website.  
It includes product display, cart system, authentication pages, and smooth UI design.

---

## ✨ Features
- 🏠 Home page with modern UI
- 🛒 Shopping cart system
- 🔐 Login & Signup pages
- 💳 Payment page UI
- 🎨 Responsive design
- 📱 Mobile-friendly layout
- 🎥 Media support (images & videos)

---

## 🧱 Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
