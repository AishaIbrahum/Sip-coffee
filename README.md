# ☕ Sip Coffee Website

A modern, fully responsive coffee shop website built using **HTML, CSS, and JavaScript**.  
Sip Coffee simulates a real café experience with smooth UI, product browsing, cart system, and checkout flow.

## ☕ About the Project

Sip Coffee is a frontend web project designed to simulate a real-world coffee shop experience.

It provides users with a smooth café-like journey where they can:
- Browse coffee menu and available products
- Add items to the shopping cart
- Review and manage selected items
- Proceed through a simulated checkout process

---

## 📌 User Journey

1. User lands on the homepage  
2. Browses coffee menu and products  
3. Selects items and adds them to cart  
4. Reviews order in cart page  
5. Proceeds to checkout (payment UI simulation)

---

## 🧾 Pages Included

- 🏠 Home Page → Café introduction and featured items  
- ☕ Menu / Products Page → Coffee selection and display  
- 🛒 Cart Page → Order summary and item management  
- 💳 Payment Page → Checkout UI simulation  
- 🔐 Login Page → User login interface  
- 📝 Signup Page → New user registration interface  
- 📞 Contact Page → Contact form for customers
